# HireSmart – Backend

The **HireSmart Backend** is the server-side implementation of the HireSmart platform.  
It handles user authentication, profile management, interview processes, and data storage for the application.

This backend is designed to support a smart hiring system where users can create profiles, participate in interviews, and receive feedback.

---

## Features

- User Authentication (Sign Up / Sign In)
- Secure API endpoints
- Profile management
- Interview session handling
- Data storage and retrieval
- Integration with frontend application
